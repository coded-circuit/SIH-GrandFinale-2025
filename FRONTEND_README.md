This is the frontend codebase of our Project. 
It is a next.js application with tailwind css and various other libraries and packages provided in package.json file.

Steps to Run the codebase in local Server
1. npm install  (will install all the dependencies).
2. Set up environment (env.example is provided).
3. npm run dev (The website will run on the local server.)
